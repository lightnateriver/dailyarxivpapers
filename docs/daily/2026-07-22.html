面向语言模型推理的选择性状态空间适配与检索

摘要总结
本文提出一族基于选择性状态空间模型（Mamba）的适配器，用于提升冻结大语言模型的多跳推理能力。作者认为传统 LoRA 对所有输入施加相同的静态更新，缺乏对 token 级和实例级状态变化的显式建模。为此提出两个互补模块：(1) MaLoRA（Mamba 调制低秩适配），让适配器的缩放因子成为依赖输入的动态函数，并在 token 之间维持循环状态，取代此前无状态调制的做法；(2) MaRA（Mamba 检索适配器），在段落间追踪跨段状态，选择与查询最相关的段落后再让语言模型生成答案。在 Qwen-2.5-7B、Llama-3.1-8B、Gemma-2-9B 三个冻结基座和 MuSiQue、2WikiMultihopQA 两个多跳推理基准上，该方法在每个设置下均超越 LoRA 基线，平均提升 6.8 F1（+10.5%），在最具挑战的设置上提升达 9.3 F1（+18.2%）。MaLoRA 在长上下文压力测试 RULER QA-2 上同样带来收益，表明状态化 token 调制的泛化能力。
核心创新点
  • 提出 MaLoRA：首个将 Mamba 选择性状态空间模型引入 LoRA 调制的方案，使适配器的逐 token 缩放因子具备跨 token 循环状态，突破了 TopLoRA 等无状态调制的表达能力上限
  • 提出 MaRA：一个轻量级检索适配器，通过 Mamba 状态跨段落传播来选择性召回上下文段落，无需外部检索器即可端到端训练
  • MaLoRA 和 MaRA 在 token 粒度和段落粒度两个层面互补，可以无缝集成在同一套框架内，共同提升推理质量
  • 实验覆盖 3 个基座模型 × 2 个推理基准的全部 6 个设置，均一致优于 LoRA 基线，消融实验证实状态化调制是主要增益来源
