摘要总结
本文提出一种可分解探针（Decomposable Probe），用于诊断少步蒸馏扩散模型的行为差异来源。传统评估仅用单一的 FID/CLIP 标量，无法区分模型的改变是来自架构、蒸馏目标还是扩散模型本身。该方法在提示编码器、去噪器输入、去噪器输出三个层面注入受控扰动（均值/方差/尺度三种模式、六种强度），基于 Bootstrap 中值 Bures W2² 选择性比率对 Inception 特征进行度量。作者在 23 个模型（5 个教师 + 18 个蒸馏学生，涵盖 SDXL、SD1.5、SD3.5、PixArt-alpha、FLUX 五种骨干家族，UNet/DiT/MMDiT 三类架构，五种蒸馏范式）上统一评估，发现三个层面对应三个可分离因子。
核心创新点
  • 提出首个可分解的诊断探针框架，将 FID/CLIP 单一标量替换为三层、三模式、六强度的扰动分析，能够独立分离扩散模型中提示编码、隐空间和分数层各自的行为贡献
  • 发现隐空间层（latent layer）是 rectified-flow 骨干网络的近二元检测器——只有 rectified-flow 模型（SD3.5、FLUX）在低中频段的选择性比率持续超过 1，epsilon-prediction 模型无一达标
  • 通过严格的消融实验（PixArt-alpha 作为 epsilon-prediction 对照）排除了 wide-T5 条件编码的混淆因素，并证明该指纹特征在对抗性蒸馏（ADD）中从教师到学生均能保留
  • 揭示了分数层（score layer）在 UNet 家族上可区分 4-step ADD 蒸馏与非 ADD 基线，且在 UNet 和 DiT 上均观测到轨迹 rollout 早期强度的分数尖峰
解决的问题 / 适用场景
适用于扩散模型社区对蒸馏学生模型进行细粒度诊断，帮助研究者区分模型行为是来自骨干架构（如 rectified-flow vs epsilon-prediction）、蒸馏范式（如 ADD vs 非对抗性蒸馏）还是提示编码设计；也可作为标准化工具用于模型选择、蒸馏方法比较和可解释性分析。
