摘要总结
针对无人/少人船舶在 Shipping 4.0 范式下压载水系统的自适应控制需求，本文提出 RL-Ballast，一种基于图神经网络的深度强化学习框架。该框架将阀门排列问题转化为 54 条可行流体转移路径（图论+深度优先搜索生成），通过帧堆叠液位观测和动作历史近似部分可观的压载环境，无需显式建模高维 POMDP。在确定性推理阶段，使用 episode 级失败动作记忆和动态动作掩码实现即时重路由，并将失败转移历史累计用于可疑阀门/管段排序，无需密集传感器支持。
核心创新点
  • 将压载水阀门排列问题转化为图论可行路径枚举（54 条），大幅降低动作空间，使 RL 可在离散小动作空间上高效学习
  • 通过帧堆叠液位 + 动作结果历史隐式近似部分可观环境，跳过显式 POMDP 建模，降低了实现复杂度
  • Episode 级失败动作记忆与动态动作掩码机制，使得确定性推理中可即时重路由、避免重复无效动作
  • 无传感器依赖的堵塞诊断方案：仅凭失败转移历史累计评分，在串行不可区分堵塞条件下达到 100% Top-3 命中率
解决的问题 / 适用场景
解决船舶压载水系统在阀门故障/管道堵塞等液压异常下的自适应路径规划和低传感器依赖诊断问题。适用于无人船、少人船等传感资源受限的航运场景，也可推广至其他流体网络（如工业管道、水处理系统）的智能控制与故障定位。

Latent Programming Horizons in Coding Agents
编程智能体中的隐式编程视界
未明确披露
点击收起
摘要总结
本文研究编码智能体底层语言模型的残差流如何内部表征被编辑程序的演化状态。通过逻辑回归探针分析隐藏状态，发现模型能够线性解码当前代码是否可解析、是否通过测试、是否减少失败测试数以及是否引入回归，在两个模型和两个基准上正确性预测 AUC 最高达 0.83。更令人惊讶的是，这些内部表征领先于智能体实际的代码编辑动作——训练探针预测未来编辑结果（在代码实际写入磁盘之前），可在多达约 25 步之前达到高于随机水平的性能，作者称之为「隐式编程视界」。
核心创新点
  • 首次揭示编码智能体语言模型的残差流线性编码了程序演化状态的多种属性（可解析性、测试通过率、回归等），AUC 最高 0.83
  • 发现模型的内部表征具有「超前预测」能力——可在实际编辑发生前约 25 步预测未来编辑结果，证明了隐式编程视界的存在
  • 探针无需重新训练即可跨基准迁移，验证了发现的泛化性和外部有效性
  • 为编码智能体的机械可解释性研究开辟了新方向，将线性探针方法引入软件工程任务分析
解决的问题 / 适用场景
解决对编码智能体内部机制缺乏理解的问题，解释 LLM 在长时间多步编程任务中如何内部表征和「预演」代码演化。适用于 LLM 驱动的编程智能体（如 SWE-agent、Codex Agent）的安全性和可解释性分析，以及对智能体行为进行提前干预和错误预测的工具开发。

CARL：面向LLM规划的约束感知强化学习

摘要总结
论文提出CARL框架，通过强化学习增强大语言模型在规划任务中对约束条件的内在感知能力，而非依赖外部工具或任务分解。CARL的核心思想是比较模型在约束输入与无约束输入下的输出分布差异，据此设计约束感知奖励函数，引导模型关注约束并惩罚忽略约束的行为。实验在BlocksWorld、TravelPlanner和T-Eval三个规划基准上验证了有效性，显著优于标准强化微调基线和先进推理模型。
