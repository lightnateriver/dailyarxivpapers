However, scaling and orchestrating such agents effectively remains challenging, due to the difficulty of coordinating parallel proof search while keeping intermediate claims organized and reliable. In this paper, we propose Danus, an orchestration system for research-level mathematical reasoning centered on a shared fact graph as a global memory-management mechanism. Danus consists of a main agent that performs planning and coordination, multiple worker agents that carry out proof search in parallel, and a stateless verifier that checks proposed mathematical claims before they are admitted into the fact graph. Each verified fact is stored together with its proof and logical dependencies, allowing the system to build long arguments incrementally while keeping the shared proof state organized. The main agent periodically summarizes the evolving proof state, redirects workers across promising directions, and supports interaction with human mathematicians through progress reports. We evaluate Danus through six research-level case studies in algebraic geometry, singularity theory, and combinatorics, illustrating how the fact-graph memory mechanism enables Danus to construct long, detailed mathematical proofs. Our results suggest that fact-graph-based orchestration provides an effective route toward scaling mathematical reasoning agents for long-horizon research problems. In applied settings, the environment usually provides sparse or delayed feedback. This makes it difficult for the model to pinpoint which actions in its reasoning led to success or failure. So, learning effectively from these signals is hard because the model must determine how each failure should inform meaningful behavioral corrections in subsequent iterations. We introduce a training framework, Self-Review Reinforcement Learning, that embeds an explicit self-review step into each RL episode. When a first-pass response fails, the model generates a self-review to identify what went wrong, which conditions an improved second attempt. Unlike inference-time reflection approaches, such as Reflexion, the framework optimizes self-review with policy gradients and internalizes improvements into the base policy via selective distillation, ensuring they persist across future episodes. A cross-episode memory keeps successful self-reviews for reuse when encountering similar tasks in future episodes during training. We evaluate SRRL against a standard RLVR baseline using the GRPO optimizer across two language models, Qwen 3-4B and OLMo-3- 7B, on GSM8K benchmark. SRRL consistently outperforms the RLVR in final reward performance and achieves greater learning efficiency by successfully transforming feedback into behavioral improvement. 摘要总结
这篇论文提出一种用于训练大语言模型的强化学习框架 SRRL，核心是在每个强化学习回合中加入显式的“自审查”步骤。当模型首轮回答失败时，会先生成对错误原因的自我复盘，再据此进行第二次改进作答。作者还通过策略梯度优化自审查过程，并用选择性蒸馏把有效改进沉淀回基础策略，同时利用跨回合记忆复用成功的自审查经验。
核心创新点
  • 提出 Self-Review Reinforcement Learning，在强化学习训练回合内显式插入自审查与二次尝试机制。
  • 不是只在推理时做反思，而是直接用策略梯度优化自审查行为，使其成为可学习的训练信号。
  • 引入选择性策略蒸馏，把自审查带来的改进行为内化到基础策略中，避免只在当前回合生效。
  • 设计跨回合记忆机制，保存成功的自审查经验并在后续相似任务中复用。

摘要总结
这篇论文关注多跳 RAG 中证据逐步获取的问题，指出现有方法虽然包含迭代检索、查询改写、证据评估和充分性判断等操作，但通常被固定在特定流水线中。作者提出 DynaKRAG，将多跳证据获取建模为对一组原子证据操作的状态条件控制：先根据当前状态生成可执行动作集合，再由学习到的控制器选择下一步操作。实验表明，该方法在 HotpotQA、2Wiki 和 MuSiQue 上都优于对比基线，说明在演化的证据状态下协调检索、诊断与面向信息缺口的获取是有效的。
核心创新点
  • 将多跳证据获取统一表述为基于状态的原子证据操作控制问题，而不是依赖固定流程或预设拓扑。
  • 设计 validity layer，根据当前证据状态动态构造可执行动作集合。
  • 引入学习式控制器，在当前有效操作中自适应选择下一步证据操作。
  • 通过充分性反馈与受控检索上限实验，揭示“更多检索并不总是更好”，强调证据控制的重要性。
